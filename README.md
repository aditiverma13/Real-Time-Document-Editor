# Real-Time Collaborative web page 

> A web-based real-time collaborative Doc 

 **Live Demo:** https://drive.google.com/file/d/1SK02BqfO_yIW936GhGoigHdbndobqncE/view?usp=sharing

---

##  Key Features

 Real-Time Collaboration between users  
 User & Document Management  
 WebSockets (Socket.IO) for real-time bi-directional communications  
 Secure Authentication with JWT  
 Conflict Handling for concurrent edits  
 Execution of code through a REPL service  
 Frontend integrated with a terminal output display  
 Project structure with backend and frontend separation

---

## Tech Stack

### Frontend
* **React**  
* **Vite** — frontend dev server  
* **Tailwind CSS** — utility-first CSS  
* **Socket.IO-client** — real-time socket connection

### Backend
* **Node.js + Express** — REST APIs  
* **Socket.IO** — WebSocket communication  
* **MongoDB & Mongoose** — database for users, sessions  
* **JSON Web Tokens (JWT)** — secure auth  
* **Body-Parser & Cookie-Parser**

### Tools
* **Nodemon** — auto restart on backend changes  
* **dotenv** — environment variable management
